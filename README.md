第279題
證券商之業務人員執行業務應本誠實及信用原則，下列何者為其本於職業道德而不得有之行為？(1)辦理受託買賣有價證券時，有隱瞞、詐欺或其他足以致人誤信之行為(2)以他人或親屬名義供客戶申購、買賣有價證券(3)接受客戶委託買賣有價證券時，同時以自己之計算為買入或賣出之相對行為(4)以上皆是 
原答案是(1)，請改為(4) 

使用說明
1. 請先下載名為金融市場常識題庫系統.py、金融市場常識題庫.json、職業道德題庫系統.py、職業道德題庫.json
2. 把他們放在同個目錄下
3. 要有python環境的電腦 (python官方連結 https://www.python.org/ )
4. 下載好python後，點選他
5. 進入python安裝介面後，記得點選Add Python x.x tp PATH
6. 安裝完python後，先進入命令提示字元，輸入pip install Pyqt6
7. 安裝完Pyqt6後點選金融市場常識題庫系統.py或職業道德題庫系統.py後，即可作答
